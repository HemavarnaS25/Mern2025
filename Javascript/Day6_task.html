Interview Questions

1. What is Git?

Git is a distributed system designed for version control to manage codebases efficiently.

2. Which command is used to make a folder or file as a GitHub repository?

The command git init initializes a folder or file as a GitHub repository.

3. Why Git is fast in executing commands?

Git operates quickly because it is implemented in the highly efficient C programming language.

4. How the staging area is called Before the first push?

Before the first push, the staging area is commonly referred to as the index or cache.

5. In a local repository, if the root folder is the repository, what is the name of the subfolder?

  1. Repository
  2. Folder
  3. Subfolder
  4. None of the above

Ans: d. None of the above.

6. What is spec file in MEAN stack?

A spec file contains test cases that validate whether a specific component, function, or module behaves as required. The term "spec" is derived from specification, outlining expected behavior.

7. If following is typed in a HTML file without its basic structure what will be there in the inspect tab?

  1. <h1>Hello world!</h1>
  2. <title>sece2025</title>
    <h1>Hello world!</h1>
  3. <h1>Hello world!</h1>
    <title>sece2025</title>

Ans: The browser structures the HTML by default, placing the <title> inside the head and the <h1> inside the body, adhering to HTML parsing rules.

1. Why Git is not showing the git commit command while adding a file and it is showing a command for unstage?

2. What is the difference between href and src and why is it used in img and anchor tags?

The href attribute is used in anchor tags for navigation to external resources, while the src attribute is used in img tags to embed external content like images within the webpage.

3. How the title tag is fetched to head and h1 is fetched to body when rendering an HTML file without structure?

When parsing an unstructured HTML file, the browser automatically assigns the <title> tag to the head section and the <h1> tag to the body for proper rendering.

4. While exporting a component, why is the default keyword mandatory?

The default keyword is optional; it enables default exports, making the component importable without curly braces. Named exports require the use of curly braces.

5. What is the difference between spread and rest operators?

The spread operator expands elements of an array or object, while the rest operator gathers multiple values into an array, often for function parameters.
