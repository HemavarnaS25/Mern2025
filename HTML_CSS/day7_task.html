Interview Questions

1. How does React's StrictMode help developers?

Answer: React's StrictMode highlights potential issues by providing additional checks and warnings for child components, helping developers improve their code during development.

2. What differentiates the spread operator from the rest operator?

Answer: The spread operator (...) unpacks elements from arrays or objects, whereas the rest operator (...) gathers multiple elements into a single array or parameter in functions.

3. Why should React components have capitalized names?

Answer: React components should start with a capital letter so they are recognized as components rather than standard HTML tags.

4. How does the browser handle an unstructured HTML file with a title and h1?

Answer: Browsers automatically place the title tag in the head section and the h1 tag in the body section by applying default HTML parsing rules.

5. What is a linting error, and why is it important?

Answer: A linting error occurs when code deviates from established coding standards or best practices. Fixing these errors enhances code quality and prevents potential bugs.

6. Why does accessing an object's property using the dot operator in a for-in loop return undefined?

Answer: It happens because the dot operator looks for a property named "key" instead of dynamically accessing the property corresponding to the variable's value.

7. Why are there placeholders (three dots) in code examples for if...else and do...while?

Answer: The three dots (...) in such examples indicate omitted logic or code that should be added in those sections as needed.

8. What is the difference between the local repository and remote repository before and after the first push?

Answer: Before the first push, there is no link between the local and remote repositories. After the first push, a connection is established.

9. Why is Git known for its fast execution of commands?

Answer: Git is fast because it is implemented in the C programming language, which ensures high performance.

10. What is the purpose of initializing a folder with Git?

Answer: Initializing a folder with Git turns it into a repository, enabling version control and Git operations.

11. What type of tree structure is used in the DOM?

Answer: The DOM uses an N-ary tree structure, where each node can have multiple child nodes.

12. How much revenue does WhatsApp generate annually?

Answer: As of late 2024, WhatsApp's estimated annual revenue is approximately $1.3 billion.

13. Why is the default keyword optional when exporting a component?

Answer: The default keyword is used to specify default exports, making the component importable without curly braces. Named exports, however, require curly braces.
